# scoop-bucket

适用于 Windows 命令行安装工具 [Scoop](https://scoop.sh) 的自定义 bucket。

## 使用方法

```bash
# 添加本 bucket 到 Scoop
scoop bucket add <bucketname> https://github.com/tianmuyun/scoop-bucket

# 从本 bucket 安装应用
scoop install <bucketname>/<appname>
```
